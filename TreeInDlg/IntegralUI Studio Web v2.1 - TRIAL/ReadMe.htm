IntegralUI Studio for Web

v2.1

IntegralUI Studio for Web is a suite of advanced AngularJS directives and jQuery widgets. All components have many features like fully customizable appearance, optimized for fast data loading and update, improved drag and drop functionality, animations, support for right-to-left layouts, themes and much more.

What's included

  • Accordion - A list of expandable groups with header and content panel
  • CheckBox - Native AngularJS directive which represents a check box
  • ComboBox - Native AngularJS directive which represents a combo box
  • ContextMenu - Adds a custom context menu to other AngularJS Directives
  • Menu - Allows you to easily navigate among pages in your web apps
  • SlideBar - Animates slides composed of images or custom content
  • TabStrip - Allows you to create tabbed user interface in your web apps
  • TreeGrid - A Grid component which shows hierarchical data structures in multiple columns
  • TreeView - Fast data loading and update of hierarchical structures

What's New

  • Use of ComboBox directive as an editor in TreeGrid cells, displays drop-down list
  • Filtering:
    • AngularJS IntegralUIFilter service provides advanced built-in filtering operations
    • Option to create custom filtering operations in your code usign the above service
    • TreeGrid and TreeView components comes with integrated filtering
    • Option to create custom filter templates
  • Sorting:
    • TreeGrid and TreeView includes option to apply custom or built-in sorting
    • Sort rows and items in ascending or descending order, or choose none to stop sorting
    • Ability to create your own custom sort operation very easily
  • Improved overall performance of the TreeGrid
  • Vertical scrolling in TreeGrid is now smooth, tooltip is no longer displayed
  • Option to show/hide TreeGrid columns
  • Option to prevent cells from editing on individual level
  • Option to prevent expanding of rows and items with double-click
  • Pressing ENTER or SPACE key over cells with a check box, now changes its value
  • New properties:
    • autoExpand, determines whether items or rows becomes expanded during drag drap
    • allowFilter, determines whether filtering will be applied
    • sorting, determines the sort order
  • New methods:
    • filter, filters the data in TreeGrid and TreeView using specified filtering parameters
    • findRowByPath, searches for a row based on specified path
    • sort, arranges the data in TreeGrid and TreeView in ascending or descending order
  • New events:
    • gotFocus and lostFocus, fired when row, item or label editor receives or loses the input focus
    • click, fired when check box area is clicked
  • and much more ...

What's Fixed:

  • Column resize no longer interferes with column hovering
  • Column reorder no longer interferes with column resize
  • During column resize, resize cursor no longer disappears when mouse cursor is outside resize area
  • Drag Drop upwards now scrolls the current view of the TreeGrid
  • Drag Drop of rows over column header and footer is now disabled
  • PageDown, PageUp handlers no longer causes scrolling of window page
  • Mouse-wheel no longer changes hover item when there is no vertical scrollbar
  • TextTrimming under FireFox no longer cuts the cell's text
  • ItemCheckedChanged is no longer fired during expand/collapse of items
  • Misfire of cellValueChanged event during control layout update is now fixed
  • If a context menu is shown for a row or item, pressing the escape key now closes the context menu
  • add/remove events are now properly fired after loadData is called
  • Fixed issue which causes right click on context menu to show the default browser context menu
  • cellClick event now fires when check box is clicked
  • Fixed the issues which causes selection of label editor text to reset the cell value
  • Fixed the issue which caused label editor not to occupy cell space in whole
  • Selecting a part of text in label editor no longer closes the editor
  • Other minor bug fixes
