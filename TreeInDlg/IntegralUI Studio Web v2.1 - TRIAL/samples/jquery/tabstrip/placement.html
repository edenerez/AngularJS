In this sample you can change on which side tabs are placed. The TabStrip widget has a property tabStripPlacement, which can accept one of these four values:

  • top - tabs are placed on top side, default
  • right - tabs are placed on right side
  • bottom - tabs are placed on bottom side
  • left - tabs are placed on left side

If the size of all tabs is longer then widget size, navigation buttons will appear, which you can use to scroll among tabs.
