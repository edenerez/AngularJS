IntegralUI Menu widget allows you to easily navigate among pages in your web app. You can create menus from HTML, locally using JavaScript or remotely using JSON data files. Each menu item can have an icon, title, custom content or can act as a separator. The content is fully customizable.

Using links in right panel, you can check out different features of Menu widget.
