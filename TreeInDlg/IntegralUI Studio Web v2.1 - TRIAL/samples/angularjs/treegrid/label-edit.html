TreeGrid / Label Edit


Activate Editor On:


In this sample you can enable/disable editing of cell label in each row of TreeGrid directive for AngularJS. By clicking on check box on the right, you can change the built-in properties:

  • labelEdit - determines whether cell label is editable
  • editorSettings - holds an object with global settings for all editors

There are two ways to start editing process:

  • By clicking on cell label using the left mouse button
  • By pressing the 'ENTER' key when cell has the input focus

When cell is clicked, the edit box will not appear immediately. Depending on activation type, the editor will show when:

  • cell is clicked - You need to press and hold the left-mouse button for a short amount of time and then release it, after which the edit box will appear
  • cell is double-clicked - teh editor will appear imediatelly
  • cell has input focus - the editor will appear when ENTER key is pressed, activation type doesn't matter in this case
