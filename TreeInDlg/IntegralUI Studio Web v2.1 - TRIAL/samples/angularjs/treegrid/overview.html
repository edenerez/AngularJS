IntegralUI TreeGrid is a native AngularJS Grid directive with high performance during data load and update even with very large data sets. You can create and modify hierarchical data structures in multiple columns on the fly, with fast data load either from local or remote data source. It is fully customizable with option to change its appearance during run-time.

Using links from sample list above, you can check out different features of TreeGrid directive.
