TreeView / Local Data Binding


In this sample we are using a local data to load the content of TreeView. In order this to work, you need to match the data fields from local data source to the fields of item variable. You can use any names for your data, as long as you match those names with item variable fields, the TreeView will be populated.

As you can see in current view above, some items are collapsed while others are expanded. This also can be set in your local data. Additionally you can set an item icon, state image, etc.

For more information check out the source code of this sample.
